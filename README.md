# Q-SmartGrid — Quantum + Generative AI for Sustainable Campus Energy

An interactive Streamlit dashboard that simulates energy usage across campus buildings, forecasts future loads, and applies AI-based optimization to reduce energy peaks, cost, and CO₂ emissions.

###  Features
- Adjustable simulation parameters (days, buildings, peak reduction)
- Forecast vs Optimized load visualization
- Energy and cost savings metrics
- Bilingual interface (English / Arabic)

### Tech Stack
Streamlit • Python • NumPy • Pandas • Cirq (Quantum Layer) • Vertex AI (Forecasting)
